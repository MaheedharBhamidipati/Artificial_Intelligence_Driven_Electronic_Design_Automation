# 📘 Basic Verilog Codes – Examples Collection

## 📌 Overview

This folder contains basic Verilog HDL example codes used for:

- Learning digital design
- Testing the AI EDA Tool
- RTL parsing verification
- Simulation practice
- Schematic generation testing
- FPGA experimentation

These examples include both combinational and sequential logic designs commonly used in VLSI and FPGA development.
