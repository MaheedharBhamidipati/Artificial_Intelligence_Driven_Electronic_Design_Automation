# ⚙️ Installations & Tool Setup

## 📌 Overview

This folder contains installation resources, setup guides, and required dependencies for running the AI EDA Tool project.

It includes software, libraries, EDA tools, and environment configurations needed for:

- RTL simulation
- Schematic generation
- AI analysis
- FPGA workflows
- Verification
- Visualization

## 🔹 Python

Used for backend processing and AI modules.

Recommended Version:

- Python 3.10 or above

## 🔹 Node.js

Used for frontend development.

Recommended Version:

- Node.js LTS Version
